# TIL とは？

TIL は、世界中の開発者の間で 10 年ほど前から行われている学習記録の習慣 です。

→「Today I Learned」の略で、その名の通り 「今日学んだこと」 を記録していきます。

GitHub を使って小さなアウトプットを継続することで、学習の過程を客観的に見える化できるのが特徴です。

さらに、GitHub に草も生え 🌱、Git / GitHub の操作にも自然と慣れるため、学習習慣としてとても相性が良い取り組みです。

## TIL（Today I Learned）の特徴

* 毎日の学び：その日に学んだことを簡潔に記録
* 小さな気づきで OK：完璧な記事である必要はない
* 知識の蓄積：自分だけの学習データベースになる
* 後から検索可能：過去の学びをすぐに振り返れる
* 記事として公開：記録をベースに、発信へ発展させられる

## TIL のメリット

1. 学習の定着：書くことで理解が深まり、忘れにくくなる
2. 活動の可視化：毎日のコミットが GitHub 上で確認できる
3. アウトプットの習慣化：気軽に発信する練習になる
4. ポートフォリオになる：学習姿勢や継続力の証明になる
5. 知識整理の力がつく：情報を構造化する思考が身につく
